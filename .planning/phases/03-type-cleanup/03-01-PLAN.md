<interfaces>
YamlValue recursive type:
  type YamlValue = string | number | boolean | null | YamlValue[] | Record<string, YamlValue>
  type FrontmatterObject = Record<string, YamlValue>

This replaces: type FrontmatterObject = Record<string, any>

Call-site impact: anywhere that reads frontmatter fields with `.field` access will need a type guard
or cast to the specific type (string, number, etc.) since YamlValue is a union.

Pattern for safe field extraction:
  function asString(v: YamlValue | undefined): string {
    return typeof v === "string" ? v : String(v ?? "");
  }

output.ts AnyValue pattern:
  Current: type AnyValue = any; function extractField(obj: AnyValue, path: string): AnyValue
  Target: use unknown + type guard:
    function isObject(v: unknown): v is Record<string, unknown> { return typeof v === "object" && v !== null && !Array.isArray(v); }
    function extractField(obj: unknown, path: string): unknown { ... }

state.ts any usage:
  Read the file to identify specific any usages, then replace with appropriate types.
  State data follows a known shape — define a StateData interface.
</interfaces>
