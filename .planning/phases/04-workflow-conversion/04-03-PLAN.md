    - Add a helper function `readVersionTag(filePath)` to src/wxp/index.ts that reads the gsd-version tag from a file and returns { v, doNotUpdate }.
    - This function will be used by Phase 5's harness copy logic to check do-not-update before overwriting.
  </behavior>
  <action>
    Add to src/wxp/index.ts:

    ```typescript
    export interface WxpVersionTag {
      v: string;
      doNotUpdate: boolean;
    }

    /**
     * Read the <gsd-version> tag from a workflow file.
     * Returns null if no version tag is found.
     * Used by harness copy logic (Phase 5) to check do-not-update flag.
     */
    export function readVersionTag(filePath: string): WxpVersionTag | null {
      try {
        const content = readFileSync(filePath, "utf8");
        const m = /<gsd-version\s+v="([^"]+)"(\s+do-not-update)?\s*\/>/.exec(content);
        if (!m) return null;
        return { v: m[1], doNotUpdate: !!m[2] };
      } catch {
        return null;
      }
    }
    ```
  </action>

    - `readVersionTag` called on a file with `<gsd-version v="1.12.4" />` returns `{ v: "1.12.4", doNotUpdate: false }`.
    - `readVersionTag` called on a file with no version tag returns `null`.
